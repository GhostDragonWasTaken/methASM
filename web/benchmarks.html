Benchmark Methodology

Test Environment

All benchmarks were run on the same hardware and operating system:

  • CPU: Intel Core i7-12700K (12 cores, 20 threads)
  • RAM: 32GB DDR4-3200
  • OS: Windows 11 64-bit
  • Compiler: Methlang 1.0, GCC 11.2, Rust 1.65, Go 1.19, .NET 6.0
  • All tests run 10 times, median value reported
  • Optimization flags: Methlang -O, GCC -O3, Rust --release, Go default
